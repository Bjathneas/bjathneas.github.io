# What are Data Structures
A **data structure** is a way of organizing and storing data, allowing for efficient access and manipulation

# Classifications
* **Linear Data Structures**: Data structures in which the data is stored linearly. Meaning the data is stored in a singulare list. *Examples: array, stack, queue, linked lists*
  * **Static Data Structures**: The data structure has a fixed memory size, meaning no change in size during runtime. *Example: array*
  * **Dynamic Data Structures**: The data structure has no fixed memory size, and its size can be changed during runtime *Examples: queue, stack, linked lists*
* **Non-Linear Data Structures**: Data structures where the data is not sored linearly. Data cannot be looped in a single loop. *Examples: trees and graphs*
